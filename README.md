# Hopfield Network and Stimuli Repetition
Unpublished work by Joshua Oon Soo Goh [\[1\]](#1) in fulfillment of Psych 591 Neural Network Modeling course conducted in Spring 2007 at the Dept. of Psychology, University of Illinois at Urbana-Champaign, IL, USA, instructed by John Hummel. This work extends the Hopfield Network as implemented by John Hopfield [\[2\]](#2) by evaluating its performance with respect to stimuli repetition. The primary code is writtern in R ([HopfieldNetwork.R](/code/HopfieldNetwork.R)), which has recently been ported to a Jupyter Notebook [binary_autoencoder.ipynb](/code/binary_autoencoder.ipynb) with a [Google Colab instance](https://colab.research.google.com/drive/1vPtrVNpzqqMQak4g5tINBOwY5dM0Pqt5?usp=sharing)
